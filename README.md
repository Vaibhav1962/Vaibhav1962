> Once upon a third year, in the kingdom of JIIT Noida, a CS undergrad decided that building toy projects was beneath him. So he fine-tuned transformers, deployed multi-model APIs, published IEEE papers, and somehow got cited within 4 months. This is his repository of conquests.

I build **ML systems that actually work in production** — not just notebooks that run once. My weapons of choice: FastAPI, RoBERTa, fuzzy logic, Docker, and an unhealthy obsession with latency metrics.

## ⚔️ Conquests & Campaigns

### 🧠 Soul — Mental Health & Emotion Prediction API
> *The Crown Jewel of the Arsenal*

- Fine-tuned **RoBERTa** for **28-class multi-label emotion classification** — 93% accuracy
- Binary crisis detection with **F1: 0.928**
- Serves predictions at **sub-200ms p95 latency** under concurrent load
- Real-time crisis alerting via async event emission
- SHA-256 input hashing, structured logging, horizontal scaling ready

### 🎓 Academic Digital Twin
> *The Grand Engineering Project*

- Engineered fuzzy control systems for **pressure estimation** and **goal-effort alignment** — 12–24 rules per system, centroid defuzzification
- K-Means for student archetypes + Random Forest for risk detection — cluster labels piped as encoded features
- What-If simulation, ablation, diminishing returns, and goal sensitivity endpoints
- Full-stack: Flask REST API → Supabase → React → Docker → Render

### 🌿 Plant Disease Classifier
> *The Field Campaign*

- Fine-tuned **MobileNetV2** on 38-class dataset — **93.53% test accuracy**
- Selective layer unfreezing + early stopping + real-time Keras augmentation
- Built for **edge-device deployment**
- Per-class confusion matrix analysis + targeted dataset rebalancing
